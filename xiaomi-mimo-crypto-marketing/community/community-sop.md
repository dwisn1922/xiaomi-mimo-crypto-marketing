# Community Management SOP

## Channels

- Telegram: real-time community, announcements, support.
- Discord: structured channels, roles, events, ambassador program.
- X/Twitter: discovery, memes, threads, KOL amplification.

## Telegram Structure

Pinned message wajib berisi:
- What is MiMo?
- Official links.
- Rules.
- Scam warning.
- Disclaimer: not financial advice.

Daily mod tasks:
- Welcome new members.
- Hapus spam/scam cepat.
- Jawab FAQ pakai template.
- Dorong diskusi harian.
- Screenshot insight penting untuk weekly report.

## Discord Structure

Recommended channels:

```text
#announcements
#start-here
#rules
#official-links
#general
#mimo-education
#meme-lab
#kol-and-partners
#support
#mod-log
#ambassadors
```

## Moderation Rules

Auto-ban:
- Phishing links.
- Impersonation admin.
- Asking seed phrase/private key.
- Fake airdrop links.

Warn first:
- Spam repeated message.
- Off-topic promotion.
- Aggressive shilling.

## Daily Engagement Prompts

- “Apa hal paling bikin bullish/bearish dari narasi MiMo?”
- “Drop meme MiMo terbaik hari ini.”
- “Pertanyaan beginner tentang wallet/security?”
- “Konten edukasi apa yang perlu kita buat besok?”

## Weekly Events

- Monday: market/community recap.
- Wednesday: education mini-class.
- Friday: meme contest.
- Sunday: AMA / townhall.

## Crisis Response

1. Pause scheduled promotional posts.
2. Pin clarification.
3. Admin only mode jika raid/scam parah.
4. Kumpulkan fakta, jangan spekulasi.
5. Post update maksimal 30-60 menit setelah isu valid.

## Admin Script Example

```text
GM MiMo fam 👋
Hari ini fokus kita: edukasi wallet safety + meme challenge.
Reminder: admin tidak pernah DM duluan, jangan klik link airdrop palsu, dan jangan pernah share seed phrase/private key.
```
